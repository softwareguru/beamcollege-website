Beam College is an online, free educational program that provides hands-on training to solve data processing use cases using <a href="https://beam.apache.org">Apache Beam&reg;</a>. 

### Topics 

Beam College 2026 sessions are organized in the following 3 tracks:

 * **Beginner (April 21st)**: Overview of what is Apache Beam and how it fits on your data processing stack, building blocks and fundamentals for building your first pipeline.
 * **Tips & Tricks (April 22nd)**: Sessions focused on solving specific use cases with Apache Beam including processing video data, processing time series data, migrating from batch to streaming, etc.
 * **New Features (April 23rd)**: Sessions focused on reviewing some of the latest features in Apache Beam covering topics like remote inference for AI, RAG pipelines, managed I/O with Iceberg, etc. 

Sessions are provided as a live online conference. We encourage participants to join live in order to maximize their learning and engagement opportunities. However, all sessions are recorded and published for future reference.
